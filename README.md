# Coding Junior - Full Stack Assignment

This project is a mini code editor built using **Next.js**.

## Features
✔ Code Editor  
✔ Run Button (eval-based execution)  
✔ Auto-Fix Button  
✔ Help Panel with keyword responses  
✔ Clear state management  

## Auto-Fix Rules
- Add missing semicolons  
- Fix indentation  
- Correct bracket issues  

## Help Keywords
- run  
- editor  
- autofix  
- help
